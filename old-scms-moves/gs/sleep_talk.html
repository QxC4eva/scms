[SHORT DESC] Picks one of the user's moves at random. Can only be used while sleeping. [LONG DESC]

Description

Picks one of the user's moves at random. Can only be used while sleeping. The following moves cannot be chosen by Sleep Talk:

  • Bide
  • Dig
  • Fly
  • Metronome
  • Mirror Move
  • Skull Bash
  • Sky Attack
  • Sleep Talk
  • SolarBeam
  • Razor Wind
  • Any move the user cannot choose for use

No PP is spent for the move used. If Sleep Talk selects Hyper Beam, Sleep Talk loses 1 PP when using the move, and if the user remains asleep next turn, Sleep Talk is forced to be selected for the recharge turn and loses another PP.

If Sleep Talk selects Rollout in consecutive turns, it is not boosted.
