[SHORT DESC] Does 1/4 damage per turn at the cost of half the user's max HP if the user is a Ghost. Boosts Atk and Def and lowers Spe 1 stage otherwise. [LONG DESC]

Does 1/4 damage per turn at the cost of half the user's max HP if the user is a Ghost. The effect is removed when the target switches. Boosts Attack and Defense one stage and lowers Speed one stage if the user is not a Ghost. If the user is a Ghost, Curse will fail if there are other Pokemon in play.
