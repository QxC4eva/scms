[SHORT DESC] Power doubles if the user's teammate fainted the previous turn. [LONG DESC]

Description

Deals damage. If a Pokemon in the user's party fainted on the turn prior to using this move, its Base Power doubles.

Competitive Use

Retaliate is marginally useful on Normal-type revenge killers such as Ambipom, Cinccino, and Stoutland. However, even though it can reach a sky-high Base Power of 140, the fact that an ally has to have fainted in the previous turn makes it a rather inconsistent move. It is better to use more reliable Normal-type moves such as Return and Double-Edge instead.

Related Moves

Return is a more consistent alternative with 102 Base Power.

Double-Edge is a more consistent alternative with 120 Base Power, but causes recoil damage.
