[SHORT DESC] Healing varies based on the number of uses of Stockpile. [LONG DESC]

Description

Heals the user. Fails if the user has not used the move Stockpile before this move. After Swallow is used, the Stockpile counter is reset, and the user loses all Defense and Special Defense boosts gained through using Stockpile. The percentage of max HP restored depends on the number of Stockpiles used before this move, as follows:

Number of Stockpiles Percentage of max HP restored
1 25%
2 50%
3100%

Competitive Use

Just about any other source of recovery is better than Swallow; abandoning the Defense and Special Defense boosts accumulated with Stockpile isn't the best of ideas. If the Pokemon in question lacks a reliable recovery option, such as Roost or Recover, try using Rest instead; after a few Stockpile boosts, your Pokemon should be able to endure the turns spent sleeping.
