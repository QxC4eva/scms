Alright, everyone, the nominees are up. We received a lot of nominations, but these were the cream of the crop. When voting, be sure to only send in one (1) nominee for each category. Please note that you do not have to vote for all the categories; vote only where you feel comfortable. All nominations should be sent directly to The Smog, along with any questions you might have. Good luck to all of the nominees!

Votes must be cast before the release of Issue 25 of The Smog. Remember that we've moved to a monthly release schedule, so be sure to send those votes in early!
