Tabla Bicho

Descripción

Aumenta el Poder Base de ataques de tipo Bicho mientras el usuario lleva este objeto. Los Pokémon con la habilidad Multitipo se vuelven de tipo Bicho al entrar en combate cargando esta Tabla.

General

En cualquier Pokémon aparte de Arceus, este objeto trabaja exáctamente igual que el Polvo Plata; para más información acerca del uso competitivo de este objeto, revisa la descripción de Polvo Plata. La única diferencia entre estos dos objetos es el Poder Base de Lanzamiento; la Tabla Bicho tiene un Poder Base de 90 cuando es Lanzada.
