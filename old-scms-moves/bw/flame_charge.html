[SHORT DESC] Boosts user's Speed by 1 stage. [LONG DESC]

Description

Deals damage and has a 100% chance to boost the user's Speed by one stage.

Competitive Use

Rarely seen, the usefulness of this move is debatable. It can be used by powerful Pokemon, such as Heatran or Chandelure, to set up a sweep, but is usually outclassed by moves such as Rock Polish and Shell Smash, which boost the Speed stat two stages. An important thing to note is that Flame Charge will not boost the user's Speed if the attack does not execute successfully.

Related Moves

Flare Blitz is a stronger physical alternative with 120 Base Power, however it causes recoil damage.
