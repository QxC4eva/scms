[SHORT DESC] Throws the user's hold item at the foe. Power and effect depend on the item. [LONG DESC]

The user throws its hold item at the target, which consumes the item. The power of the move depends on the item. Fling has no interaction with Sticky Hold. Even if Fling fails to do any damage the item being Flung is lost. Fling fails if Embargo is in effect for the user, if the user isn't holding an item, or if the user is holding an item which doesn't appear in the following list.
