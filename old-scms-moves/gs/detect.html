[SHORT DESC] Prevents anything from working on the user that turn. [LONG DESC]

Description

Prevents any move from working on the user that turn. Perish Song hits through Detect. Consecutive uses lower the chance of working by 50%, so the first use has a 100% success rate, and if it is used the following turn again, it has a 50% chance to work, and then it has a 25% chance to work.
