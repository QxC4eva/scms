[SHORT DESC] Repeats for 2-3 turns. Confuses the user at the end. [LONG DESC]

Locks the user into this move for 2-3 turns. Confuses the user at the end.

Related Moves

Thrash is a Normal-type equivalent.
Outrage is a Dragon-type equivalent, but with higher Base Power.
Uproar is a Normal-type equivalent, but with lower Base Power and prevents any Pokemon in play from falling asleep.
