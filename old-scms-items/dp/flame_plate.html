[SHORT DESC] Raises power of Fire-type moves 20%. Pokemon with Multitype become Fire-type. [LONG DESC]

Description

Raises the power of the holder's Fire-type moves 20% when this item is held. Pokemon with Multitype become Fire-type when switching in while holding this item.

Competitive Use

On any Pokemon other than Arceus, this item performs almost exactly like Charcoal; for information on the competitive use of this item, see the Charcoal description. The only difference between these two items is the Fling Base Power; Flame Plate has 90 Base Power when Flung.
