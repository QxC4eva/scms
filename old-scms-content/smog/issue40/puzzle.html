Parameter Puzzles

Your challenge for these questions is to obtain the given list of Pokémon from the "/dexsearch" command on Pokémon Showdown! using the correct number of parameters as shown! Note that "all" does not count as a parameter and that the exclusions mentioned are included in the total number of parameters. Can you figure out which parameters were used to obtain these results?

Simply type "/dexsearch" on Pokémon Showdown! for a quick introduction to the command. The first three users to send in all 15 correct answers will earn themselves a custom title on the Smogon forums, failing which the three users with the highest scores will earn these custom titles instead!

The fourth level of parameter puzzles is a recent addition, so I guess an explanation is in order. What these puzzles do is combine multiple dexsearch results into a single list, requiring you to reverse engineer the sets and the parameters used to define those sets. For example, suppose I had a puzzle that looked like this:

  • Aipom, Furret, Pachirisu, Pikachu, Raichu, Raticate, Rattata, Sentret, Smeargle (2 sets, totaling 4 parameters)

Eventually, I'd find that the answer would be a combination of {Follow Me, Nuzzle}, which gives Pachirisu, Pikachu, Raichu, and Smeargle, and {Run Away, U-turn}, which gives Aipom, Furret, Pachirisu, Raticate, Rattata, and Sentret. Note that even though Pachirisu appears in both sets of results, it is only shown once. Furthermore, the parameters don't have to be split evenly between the sets—you could see puzzles combining a set of 1 parameter and a set of 3 parameters! Good luck—you'll need it!
