Information

Points are updated as fast as possible after Opens are completed. Please be patient if it takes a day or two. If you notice an error in the points, contact Vinc2612 via a forum private message describing the discrepancy and linking to every Smogon Grand Slam tournament thread where the user in question earned points. Please read the Smogon Grand Slam rules very carefully before sending a PM.

Grand Slam Standings

Before submitting a complaint that the displayed score is incorrect, please be sure that you completely understand exactly how the scoring system works since it can be somewhat complicated if you have not reviewed it in detail. Mistakes are infrequent, but we are always happy to resolve them if you bring them to our attention in a friendly manner.
