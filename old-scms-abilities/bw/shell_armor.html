[SHORT DESC] Prevents critical hits. [LONG DESC]

Description

The wielder cannot be struck by critical hits.

Competitive Use

A pretty neat ability for bulky sweepers, but outside of that it is pretty worthless, since critical hits don't occur often enough to take advantage of it. Some Pokemon will use it though; Torterra will probably use it if you see it, since it will be more useful than Overgrow. Lapras could use it on a bulky set, but really Hydration is always better due to Drizzle being everywhere. It's not a very good ability but it takes an element out of the game that annoys everyone.
