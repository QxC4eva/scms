[SHORT DESC] Power increases as PP decreases. [LONG DESC]

If Trump Card's PP after use is:

  • 4+, it has 40 Base Power
  • 3, it has 50 Base Power
  • 2, it has 60 Base Power
  • 1, it has 80 Base Power
  • 0, it has 200 Base Power

If Trump Card is selected by another move (for instance, Sleep Talk), it uses the remaining PP of the move that chooses it, rather than the remaining PP of Trump Card, to determine the power.

Trump Card ignores all accuracy and evasion modifiers.
