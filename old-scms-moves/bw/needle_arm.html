[SHORT DESC] 30% chance to flinch the target. [LONG DESC]

Description

Deals damage and has a 30% chance to flinch the target.

Competitive Use

There is almost no reason to use Needle Arm competitively, as it's outclassed by more powerful Grass-type moves, and despite a decent chance to flinch the opponent, this alone is not enough reason to use it instead of another more powerful options like Seed Bomb or Leaf Blade. So there is no recommend situation for using this move.

Related Moves

Seed Bomb is a more powerful option that is better for any physical Grass-type attacker.

Wood Hammer is a much more powerful option that has recoil damage as a drawback.

Bite is a Dark-type equivalent.
