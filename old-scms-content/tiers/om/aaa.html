What is Almost Any Ability?

The idea behind Almost Any Ability is quite simple; Pokémon gain access to almost any ability! The banlist will be posted below. These abilities are accessibile to any Pokémon and are used instead of their already exsisting ability. This create many great tools for offense and defense; for example, offense has access to abilities like Adaptability and Tinted Lens, while defense has access to abilities like Regenerator and Magic Guard. This metagame is an exciting one and is able to be played on the main Pokémon Showdown! server as a permanent ladder.

Rules:

  • Aerilate, Arena Trap, Contrary, Fur Coat, Huge Power, Imposter, Parental Bond, Protean, Pure Power, Shadow Tag, Simple, Speed Boost, and Wonder Guard are banned on Pokémon that are unable to naturally have these abilities.
  • Archeops, Kyurem-B, Regigigas, Shedinja + Sturdy, Slaking, Smeargle + Prankster, Weavile, Keldeo, Bisharp, Mamoswine, and Terrakion are banned.
  • Aegislash, Greninja, Blaziken, Deoxys-D, Deoxys-S, Genesect, and Shaymin-Sky are unbanned.
