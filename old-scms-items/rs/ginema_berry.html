[SHORT DESC] Removes stat decreases. Consumed after use. [LONG DESC]

Description

If any of the holder's stats are decreased any number of stages, Ginema Berry will immediately negate that stat decrease. This effect will occur regardless of which player's Pokemon caused the decrease. This item is consumed after use. If this item is Tricked onto a Pokemon with a stat drop, the stat drop is immediately negated. Only available with an e-Reader. When a new card is scanned the new Berry overwrites any old Berry.
