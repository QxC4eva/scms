Tabla Pradal

Descripción

Aumenta el Poder Base de ataques de tipo Planta mientras el usuario lleva este objeto. Los Pokémon con la habilidad Multitipo se vuelven de tipo Planta al entrar en combate cargando esta Tabla.

General

En cualquier Pokémon aparte de Arceus, este objeto trabaja exáctamente igual que el Incienso Floral; para más información acerca del uso competitivo de este objeto, revisa la descripción de Incienso Floral. La única diferencia entre estos dos objetos es el Poder Base de Lanzamiento; la Tabla Pradal tiene un Poder Base de 90 cuando es Lanzada.
