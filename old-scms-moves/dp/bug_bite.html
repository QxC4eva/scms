[SHORT DESC] Uses foe's Berry. [LONG DESC]

If the target is holding any kind of Berry, the user takes it and uses it immediately. If the target is holding a Jaboca or Rowap Berry, the Berry has no effect, but is still removed. If the target is holding a Tanga Berry, it activates and reduces the damage of Bug Bite before it can be stolen. If an Embargoed Pokemon uses Bug Bite on a Pokemon holding a Berry, the Berry is destroyed without being activated.
