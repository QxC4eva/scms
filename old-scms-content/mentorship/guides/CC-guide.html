[Title] A guide for Contributing to Smogon [Head] [Page]
By Minus.

Introduction

This page will basically be a compilation of helpful guides for contributing to Smogon through Contributions & Corrections, The program on Smogon where all of the on-site content comes from. While C&C can seem intimidating at first, with proper guidance, it will become easier to understand and navigate.

Analysis Writing

Writing an analysis in C&C required a user to be familiar with all of the thread tags and procedures, something that is hard for newer users to become accustomed to.The following guides will help you understand the different processes involved when creating an analysis in C&C.

  • Generation 6 Analysis Format: While this isn't necesssarily a guide, it describes the purpose of each section on an analysis, like the Overview, Set Details, and Other Options.

  • Welcome to C&C--A Beginner's Guide to Contributing: This guide details each of the thread tags, WIP, Quality Control, Copyediting, and Done, and well as an overview of the Technical Projects subforum.

  • Smogon Writing 101 - Getting Started and Writing Tips: This guide focuses mainly on the writing itself rather than the actual process. It's usefull for when you're not sure about your writing.

  • Contributions & Corrections: Where the Magic Happens: While it's a little dated, this article still serves its purpose in helping new users understand the basic structure of C&C.

Grammar Prose Checking

Once the content has been revised over and over to be the best it can be, it's time for the Grammar-Prose team to come in. The GP checks the grammar of everything that goes on-site so that everything can be easily understood by the readers.

  • Spelling and Grammar Standards: Arguably one of the most important threads in all of C&C, this thread provides a universal standard in which all analyses have to be written in checked. This way, everything can be in an orderly fashion.

  • Check like the Prose: A Guide to G/P and its sequel: Both of these guides contain tips on how to create a grammar check that is thourough and fixes almost all of the mistakes.

  • On Making Subjective GP Changes: This guide explains the art of making tricky subjective edits and when to use them. This is something that should be read if you want to check at a higher level.
