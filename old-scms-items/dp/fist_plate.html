[SHORT DESC] Raises power of Fighting-type moves 20%. Pokemon with Multitype become Fighting-type. [LONG DESC]

Description

Raises the power of Fighting-type moves 20% when this item is held. Pokemon with Multitype become Fighting-type when switching in while holding this item.

Competitive Use

On any Pokemon other than Arceus, this item performs almost exactly like Black Belt; for information on the competitive use of this item, see the Black Belt description. The only difference between these two items is the Fling Base Power; Fist Plate has 90 Base Power when Flung.
