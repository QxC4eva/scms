[SHORT DESC] Varies in power depending on the number of uses of Stockpile. [LONG DESC]

Description

Deals damage. Fails if the user has not used the move Stockpile before this move. After Spit Up is used, the Stockpile counter is reset, and the user loses all Defense and Special Defense boosts gained through using Stockpile. Base Power depends on the number of Stockpiles used before this move, as follows:

Number of Stockpiles Base Power
1 100
2 200
3300

Competitive Use

If you're using Stockpile, you're likely using it to boost your Pokemon's defenses in the same vein as Cosmic Power, and not for Spit Up or its equally terrible brother Swallow. The huge Base Power of Spit Up after three Stockpile boosts might sound alluring, but keep in mind that you spend three turns gaining the boosts, and could have dealt more damage by just using any STAB attack.
