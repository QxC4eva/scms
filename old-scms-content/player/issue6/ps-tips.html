Introduction

Have you ever wanted to find out how users are able to auto-join PS! room or get those amazing colourful backgrounds? In this article, that is precisely what I'll be showing you how to do. I'll be going through an easy step-by-step guide that will allow you to have both the "Wi-Fi" and "Weather" backgrounds, as well as be able to auto-join rooms on PS!, something that I know has been requested and asked for a fair amount of times. Before we start, I'd just like to mention that I've only tried this on Google Chrome, so I'm not sure if it works other browsers. Let's begin!

Auto-join

You'll need to download something before you begin following this guide: Tampermonkey.

Tampermonkey

Tampermonkey is an extension for your browser that works as a userscript manager. Google Chrome also supports userscripts, but Tampermonkey might use scripts that are incompatible with Google Chrome.

Steps to getting Auto-join

  1. Download Tampermonkey Stable (Make sure that the icon is the black one.).

  2. Reopen your browser after the download is completed, and the Tampermonkey icon should appear in the top right corner next to the URL box.

  3. Click the Tampermonkey icon and enable it if it hasn't been already.

  4. After that, you click 'Add a new script...' and it should bring up the page below.

  5. Delete '// ==UserScript==' and everything underneath it so you have a clear box.

  6. Paste the script, which you can find here, into the empty space.

  7. This is where the settings that you specifically want come in!

    Find where the settings shown in the picture below are, and change them accordingly to how you want it. This could be from deleting some lines to changing 'scavengers' to 'overused' or 'mafia' (please make sure that when changing the room, it is in all lowercase instead of in capital letters).

  8. When completing the setting that you want, click the save button.

  9. You should then tick the box, click ' --Please choose an option-- ', scroll down to 'Enable', and press 'Start'.

  10. You should then close your current PS! tab, and when opening it again, you should have autojoin!

Backgrounds

To get backgrounds you'll also need to download an extension similar to Tampermonkey. This one is well-known throughout the server already however; it's called Stylish.

Stylish

Similar to Tampermonkey, Stylish is an extension; it can be used to make style sheets that can be enabled to customize the appearance of a web page. When using it for PS!, we're going to be installing a few themes.

Instructions

  1. Download Stylish and reopen your browser. If you have Tampermonkey installed, it should appear right beside it; if not, it should just be beside the URL.

  2. Go back to Pokémon Showdown! and click the Stylish icon.

  3. From there, you'll want to click 'Manage installed styles.', and Stylish should bring up the Stylish interface. Please take into account that these next steps are only for themes, for which you need to copy and paste the code. For most of them you can just click 'Install'.

  4. Next up is clicking 'Write new style', and this is where your own customization comes into place.

  5. Make sure to put 'http://play.pokemonshowdown.com/' in the URL box; naming your themes can really help if you ever decide to switch what theme you're using.

  6. Once you find the theme that you like, paste the code into the box and click the save button.

  7. Go to Pokémon Showdown! and make sure that you have clicked 'Enable' on Stylish for the theme that you currently want to use.

  8. Celebrate because you now have more background options! I originally found out about Stylish through this video, so a huge shoutout to TheCanadianWifier.
