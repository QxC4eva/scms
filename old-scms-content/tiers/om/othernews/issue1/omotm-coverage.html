Other Metagame of the Month — Classic Hackmons

Deoxys-Speed @ Leftovers
Ability: No Guard
EVs: 248 HP / 252 Atk / 252 Def / 252 SpA / 252 SpD / 252 Spe
Timid Nature
- Substitute
- Hypnosis
- Sheer Cold
- Gastro Acid

This set is one of the most common in the Hackmons metagame and its primary goal is to OHKO everything with never missing Sheer Cold. This works exceptionally well on Deoxys-Speed because it's extremely fast and can thus outspeed practically everything and hit them before they can even move. Hypnosis puts Pokemon that are potentially immune to Sheer Cold, such as Wondr Guard Pokemon, and then allows Deoxys-Speed the time to use Gastro Acid and nullify the ability. This set is a massive pain to deal with and must be accounted for when teambuilding.

Giratina @ Leftovers
Ability: Magic Bounce
EVs: 248 HP / 252 Atk / 252 Def / 252 SpA / 252 SpD / 252 Spe
Bold Nature
- Substitute
- Curse
- Recover
- Baton Pass

This set is able to steal the momentum of an opponent trying to set up on Giratina. Giratina uses Curse on them and forces them to hard switch out while it sets up a Substitute so Giratina is able to then pass to a Mold Breaker. Curse is also a great way to deal damage to against opposing Wonder Guards because it is effectively typeless. Giratina is a very powerful defensive threat and can shuffle a lot of the metagame.

Winning metagame featurette

Classic Hackmons is a metagame idea that has come from in-game hacking, but was not directly made by anyone in particular. Zarel did create the name, however, so the name credits go to him! Besides this, Classic Hackmons is derived from Hackmons — an older metagame with basically the same rules. In this format, anything goes! Any Pokemon can have any move, any ability, and have all stat with 252 EVs invested. There are no clauses put in place and a Pokemon may start in a Mega Evolution / Primal form without requiring its respective stone or orb. This leads to some insane strategies that include Wonder Guard and... well, countering Wonder Guard! Some top threats include Deoxys-Speed, Giratina, and Mega Gengar. This format is much more diverse than one would expect and offers great fun!
