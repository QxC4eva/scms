[SHORT DESC] Boosts Def 1 stage. [LONG DESC]

Description

Boosts user's Defense one stage.

Competitive Use

There is not much competitive value in this move. It's outclassed by other moves that raise the Defense stat faster, like Barrier. In short, there is literally no situation in which you could ever use this move.

Related Moves

Barrier is a Psychic-type equivalent that raises Defense 2 stages.

Acid Armor is a Poison-type equivalent.

Reflect doubles the Defense stat up to 1024 Defense instead of 999.
