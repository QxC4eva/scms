[SHORT DESC] Raises power of the holder's Dark-type moves 20%. Pokemon with Multitype become Dark-type. [LONG DESC]

Description

Raises the power of the holder's Dark-type moves 20% when this item is held. Pokemon with Multitype become Dark-type when switching in while holding this item.

Competitive Use

On any Pokemon other than Arceus, this item performs almost exactly like Blackglasses; for information on the competitive use of this item, see the BlackGlasses description. The only difference between these two items is the Fling Base Power; Dread Plate has 90 Base Power when Flung.
