Introduction

Cette page a pour vocation de vous permettre de comprendre comment est construite une page de Pokédex sur Smogon. Avec les informations disponibles ailleurs sur le site, notamment dans la partie FR, vous pourrez aisément utiliser tous les sets présents sur le site, et ce pour tous les Pokémon. Bonne navigation !

HP： Points de vie
ATK:　Attaque
DEF:　Défense
SpA: Attaque spéciale
SpD: Défense spéciale
Spe:　Vitesse

La somme de ces différentes caractéristiques donne un total appelé "Base stats". Par exemple, Haxorus ici présent possède 540 Base stats. (Vous pouvez vérifier)

Name: Nom. Ici, c'est Haxorus
Item: Objet. Ici c'est Lum Berry/Draco Plate
Nature: Nature. Ici, c'est、Adamant
Ability: Capacité. Ici, c'est Mold Breaker
Moveset: Les attaques utilisées. Ici : Dragon Dance, Swords Dance, Outrage, Earthquake
EVs: Les EVs utilisés pour ce set
IVs: Les IVs utilisés. En général ils sont par défaut 31 partout, sauf si c'est précisé.

Comme vous pouvez le constater, un Pokémon peut avoir plusieurs sets. C'est le cas la plupart du temps. Le protocole est identique pour chaque set, il suffit de lire de la même façon les informations données.
