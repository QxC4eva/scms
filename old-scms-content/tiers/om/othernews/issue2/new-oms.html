  • Stockpiling — by: Chopin Alkaninoff
    • Non-damaging moves cause a "Stockpile-like" effect which raises the Defense and Special Defense of the Pokemon by one. The counter caps at 3, and when it's reset the defense raises are dropped to zero. Attacking moves reset this counter.

  • Oncoming Storm — by: Cobalt314
    • Pokemon receive a Gale Wings-like ability corresponding to their primary typing.

  • League of the Phoenix — by: Adrian Marin
    • A counter is put in place to determine when a Pokemon faints, and once this timer reaches 0 the user with the higher counter wins.

  • Powered Up — by: NOCturnal Hunter
    • All abilities have their effects doubled in strength.

  • The Negative Metagame — by: Rhythms
    • Stats are replaced with a new forumla: (New Base Stat) = 150 - (Old Base Stat).

  • Hawluchange — by: MegaGallade
    • Moves receive dual-typing effects, like Hawlucha's signature move: Flying Press.

  • Recoilmons — by: Kadabrium
    • All moves cause recoil, with a new forumla that can be found in the thread.

  • Basic Elements — by: DinaIsha
    • All types are reverted to Fire, Grass, or Water, along with all existing moves of these types.

  • Mix-and-Mega — by: Flame User
    • All Pokemon are able to use any other Mega Stone and receive the typing, ability, and stat increase that this Stone provides.

  • STABmons+ — by: Eevee General
    • STABmons+ is a tier above STABmons that plays in an Ubers environment. Form changes and generation jumping is now allowed.

  • Inheritance — by: Snaquaza
    • Pokemon are able to "inherit" moves and abilities from any other Pokemon, but then may not use their own movepool / abilities.

  • Mergemons 3.0 — by: simsims800
    • The sequal to Mergemons and Mergemons 2.0, this metagame has every fully evolved Pokemon grouped with two other Pokemon which they can share abilities and moves with.

  • Burning 'Mon — by: Jajoken
    • All Pokemon enter the field burned and will be reburned instantly should it be removed via Heal Bell.

  • Base Power Damage — by: Temporaryanonymous
    • All moves deal their set number of damage. For example, High Jump Kick will deal a set 130 damage.

  • MegaLess — by: Tarantos
    • No Mega Evolutions are allowed to be used and all Mega Stones are banned.

  • Brackets — by: Snaquaza
    • Pokemon usually receive priority brackets based off of the higher Speed, but in this metagame, a Pokemon with a speed stat of 151-300 will automatically have its moves boosted to +1 priority, and so on.
