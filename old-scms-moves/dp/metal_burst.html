[SHORT DESC] Counters foe's attack with 1.5x damage, but must go last. [LONG DESC]

If the last attack to hit the user was from an opposing Pokémon, this attack deals 150% of the damage dealt to the user back to that opposing Pokémon. This move only works if the user goes after the attacker. Although a physical move, this move does not make contact with the opponent.

Related Moves

Counter is a Fighting-type equivalent, but affects only physical attacks, and has a negative priority.
Mirror Coat is a Psychic-type equivalent, but affects only special attacks, and has a negative priority.
