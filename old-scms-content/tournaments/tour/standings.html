Information

Points are updated as fast as possible after Sunday tours complete. Please be patient if it takes a day or two. If you notice an error in the points, contact Eo Ut Mortus via a forum private message describing the discrepancy and linking to every Smogon Tour tournament thread where the user in question earned points. Please read the Smogon Tour rules very carefully before sending a PM.

Tour Standings

Before submitting a complaint that the displayed score is incorrect, please be sure that you completely understand exactly how the scoring system works since it can be somewhat complicated if you have not reviewed it in detail. Mistakes are infrequent, but we are always happy to resolve them if you bring them to our attention in a friendly manner.
