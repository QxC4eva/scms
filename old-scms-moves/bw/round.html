[SHORT DESC] Power doubles if a teammate uses Round in the same turn. [LONG DESC]

Description

Deals damage. If there are other active Pokemon that chose this move for use this turn, those Pokemon take their turn immediately after the user, in Speed order, and this move's power is 120 for each other user. Pokemon with the ability Soundproof are immune.

Competitive Use

Round, the new universal TM, is a giant slouch in the effectiveness department. While it does have the chance to hit incredibly hard in doubles and triples, it serves no use at all over many more powerful attacks in singles battles, with a constantly low 60 Base Power.

Related Moves

Hyper Voice is a stronger alternative with 90 Base Power.

Return is a physical competitive alternative with 102 Base Power.
