[SHORT DESC] Varies in power depending on the number of uses of Stockpile. [LONG DESC]

Has 100 power if the user has used Stockpile once, 200 power if the user has used Stockpile twice, or 300 power if the user has used Stockpile thrice. After using Spit Up, all Stockpiles are lost. Spit Up always rolls 100/100 damage in the random damage part of the damage formula, meaning it does max damage.
