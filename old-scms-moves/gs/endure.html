[SHORT DESC] The user always survives with at least 1 HP. [LONG DESC]

Description

No attack will faint the user. If an attack would have done enough damage to cause the user to faint, it instead leaves them with 1 HP. Consecutive uses lower the chance of working by 50%, so the first use has a 100% success rate, and if it is used the following turn again, it has a 50% chance to work, and then it has a 25% chance to work.
